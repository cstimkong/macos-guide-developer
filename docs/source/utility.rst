用户态工具
===========

.. _installation:

命令 `ls`
------------

To use Lumache, first install it using pip:

.. code-block:: console

   (.venv) $ pip install lumache

命令 `xattr`
----------------

macOS 支持文件的扩展属性 (一系列 Key-value 对), 对应 Linux 中的 `attr`_ 系列工具 ( ``attr``, ``getfattr``, ``setfattr`` ) 所提供的功能。

.. _attr: https://savannah.nongnu.org/projects/attr
